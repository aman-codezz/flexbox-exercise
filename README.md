# Responsive Card Grid with Flexbox

## Description
A responsive card layout that adapts to different screen sizes using Flexbox.

## Features
- 3 columns on desktop
- 2 columns on tablet
- 1 column on mobile
- Hover effects on cards
- Images with proper aspect ratio

## Technologies
- HTML5
- CSS3
- Flexbox

## Responsive Breakpoints
- Desktop (769px+): 3 columns
- Tablet (451px - 768px): 2 columns
- Mobile (max 450px): 1 column

## How to Use
1. Open `index.html` in browser
2. Resize browser window to see responsive behavior

## What I Learned

- Media queries for responsive design
- CSS calc() for dynamic widths
